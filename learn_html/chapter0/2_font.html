Set Font Size

You can set font size using size attribute. The range of accepted values is from 1(smaller)
to 7(larger).The default size of a font is 3.

font size ="1"
font size ="2"
font size ="3"
font size ="4"
font size ="5"
font size ="6"
font size ="7"

Relative Font Size

You can specify how many sizes larger or how many sizes smaller than the preset font
size should be. You can specify it like font size="+n"or font size="-n"

font size="-1"
font size="+1"
font size="+2"
font size="+3"
font size="+4"

Setting Font Size
