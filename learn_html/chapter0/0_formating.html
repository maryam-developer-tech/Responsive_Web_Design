The HTML "pre" Element

The html "pre" element defines preformatted text.
The text inside a "pre" element is displayed in a fixed-width font(usually courier),
and it preserves both spaces and line breaks:

       My Binnie lies over the ocean.
       My Binnie lies over the sea.
       My Binnie lies over the ocean.
       Oh. bring back my Binnie to me.

HTML "q" for Short Quotations

The HTML "q" element defines a short question.
Browsers usually quotation marks around the "q" element.

WWF's goal is to:Build a future where people live in harmony with nature

HTML "blockquote" for Short Quotations

The HTML "blockquote" elements defines a section that is quoted from another source.
Browser usually indent "blockquote" elements:

Here is a quote from WWF's Website:

From 28 years, PCC has been giving a good information about IT to Student.

HTML "abbr" for Short Quotations

The HTML "abbr" element defines an abbreviation or an acronym.

The PCC based in 1989.

HTML "address" for Short Quotations

The HTML "address" element defines contact infoemation (author/owner) of a document or an article.
The "address" element is usually displayed in italic.
Most browser will add a line break before and after the element.

The HTML "cite" element defines the tile of a work.

HTML Table

An HTML table can be define with the "table" tag.
Each table row is defined with the "tr" tag.
A table header is defined with the "th" tag.
By default, table heading are bold and centered.
A table data/cell is defined with the "td" tag.
