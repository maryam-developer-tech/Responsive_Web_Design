HTML Form Control

There are different types of form controls that you can use to collect data using HTML form:

  1. text input controls
  2. password input controls
  3. checkboxes controls
  4. radio box controls
  5. select box controls
  6. file select boxes
  7. clickable buttons
  8. submit and reset button

Text Input Controls

There are three types of text input on forms:
1.Single-Line text input controls - This control is used for ite of user input, such as
search boxes or name. They are created using HTML "input" tag.
2.Password input controls - This is also a single-line text input but it masks the character as
soon as a user enter it. They are also created using HTML "input" tag but attributr is to set password.
3.Multi-line text input controls - This is used when the user is required to give details that
may be longer than a single sentence. Multi-line input controls are created using HTML "textarea" tag.

Radio Button Control

Ratio button is used when out of many options, just one option is required to be selected.
They are also created using HTML"input" tag but type is set to radio.

Select Box control

A select box, also called drop down box which provides option to list down various options
in the form of drop-down list, where a user can select one or more options.

File Upload Box

If you want to upload a file to your website, you will need to use a file upload
box, also known as a file select box. This is also created using the "input" element
but type attribute is set to file.

Button controls

There are various ways in HTML to create clickable buttons. You can also create a clickable button
using "input" tag by setting its type attribute to button.The type attribute can take the
following values: submit/description
submit : This creates a button that automatically submits a forms.
reset : This creates a button that automatically resets form control to theie initial values.
button : This creates a button that is used to tigger a client-side script when the user clicks that button.
